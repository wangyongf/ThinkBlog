Bug 反馈

感谢对 ThinkBlog 的关注和支持，如遇到 Bug 或者使用问题，可以通过以下途径反馈给我们：

  1. 在 GitHub 项目主页提交 Issue。
  2. 在 GitHub 项目主页提交 Issue。

反馈注意事项

为了能最准确的传达所描述的问题， 建议你在反馈时附上演示，方便我们理解。
